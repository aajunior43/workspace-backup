# Registro de Refeições

Uma pasta para salvar todas as refeições, dia a dia.

## Formato

Cada dia ganha seu próprio arquivo `.md` com nome `YYYY-MM-DD.md`.

```markdown

- Observações:

## Jantar
- O que comeu:

## Ceia (se houver)
- O que comeu:

- Observações:

## Resumo do dia

- Suplementos:
- Outros:
```
