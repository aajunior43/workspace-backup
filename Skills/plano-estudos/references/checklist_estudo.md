# Checklist de Acompanhamento de Estudos

## Diário
- [ ] Estudei hoje? (mínimo 1h)
- [ ] Fiz anotações?
- [ ] Resolvi exercícios?
- [ ] Revisei conteúdo anterior?

## Semanal
- [ ] Cumpri a meta de horas?
- [ ] Finalizei o tópico da semana?
- [ ] Fiz simulado/revisão?
- [ ] Identifiquei dúvidas?

## Mensal
- [ ] Terminei o módulo?
- [ ] Fiz prova/simulado completo?
- [ ] Analisei erros?
- [ ] Ajustei cronograma?

## Antes da Prova
- [ ] Todo conteúdo revisado
- [ ] Simulados feitos
- [ ] Erros analisados
- [ ] Datas memorizadas
- [ ] Legislação atualizada

---

> 💡 Copie este checklist para cada plano de estudos.
